# Bypass-YouTube-Ads
**Tampermonkey Script to Bypass YouTube Ads**

## How it works (The URL Trick)
A simple way to bypass YouTube ads is to insert a hyphen "-" between "`yout`" and "`ube`" in the video URL but you'll can see the video only (no comments, descriptions...). You can test this manually by yourself on any video:

**Original:** `https://www.youtube.com/watch?v=jNQXAC9IVRw`
**Bypass:** `https://www.yout-ube.com/watch?v=jNQXAC9IVRw`

**This script automates the process for you.**
